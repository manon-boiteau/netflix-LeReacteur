## 🗝 Installation and usage

---

Simple project, simple installation.

### 🚙 Running the project

1️. Clone this repository

`git clone https://github.com/manon-boiteau/netflix-LeReacteur.git`

`cd netflix-LeReacteur`

2️. Install packages

`npm install`  
or  
`yarn`

3️. When installation is complete:

`yarn start`
